# @add-seo-links

向SEO导航站添加新链接的标准化流程。

## 快速使用

输入 `@add-seo-links` 然后提供链接列表：

```
网站收集
- 工具地址：
  - https://example.com/ 工具描述
- 域名注册：
  - https://domain.com/ 域名工具
```

## 执行流程

1. **准备分支** - 创建Git分支
2. **添加链接** - 更新index.html
3. **质量检查** - 检查重复和格式
4. **提交和合并** - 提交到main并创建release
