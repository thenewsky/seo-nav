# 规则2: 添加链接到index.html

## 目标
将用户提供的新链接添加到index.html的合适位置。

## 执行步骤

### 2.1 理解链接分类
现有分类模块（按优先级排序）：
1. **SEO社区/学习资源** - 社区、论坛、学习平台
2. **SEO工具/关键词研究** - 关键词研究、趋势分析工具
3. **SEO工具/网站分析** - 网站流量、数据分析工具
4. **SEO工具/竞争分析** - 竞争分析、关键词难度工具
5. **SEO教程/最佳实践** - SEO学习资源、教程
6. **域名工具** - 域名查询、注册相关
7. **AI工具** - AI相关工具
8. **开发工具** - 开发辅助工具

### 2.2 链接分类规则
根据链接功能选择分类：
- **社区/论坛** → SEO社区/学习资源
- **关键词工具** → SEO工具/关键词研究
- **流量分析** → SEO工具/网站分析
- **竞争分析** → SEO工具/竞争分析
- **教程/学习** → SEO教程/最佳实践
- **域名相关** → 域名工具
- **AI相关** → AI工具
- **开发辅助** → 开发工具

### 2.3 添加链接格式
```html
<td><a href="https://example.com" target="_blank">显示名称</a></td>
```

### 2.4 命名规范
- 使用简洁明确的中文名称
- 同一网站的不同页面应区分（如：`SEO Box` 和 `SEO Box关键词`）
- 工具入口可标注功能（如：`UE (Semrush/Similarweb)`）
- 避免过长名称（建议不超过15个字符）

### 2.5 位置选择规则
1. **优先填充空单元格**：查找目标分类中的 `<td></td>` 空单元格
2. **如无空位**：在分类的最后一行添加新单元格
3. **调整rowspan**：如果添加了新行，需要更新分类标题的 `rowspan` 属性

### 2.6 表格结构维护
- 每行必须包含4个 `<td>` 单元格
- 空单元格使用 `<td></td>`
- 分类标题的 `rowspan` 值 = 该分类的行数
- 保持表格结构完整和美观

### 2.7 特殊处理
- **ROI计算器**：如果提到"提示词ROI计算器"，更新现有链接的描述
- **重复链接**：如果链接已存在，跳过或更新描述
- **模块顺序**：SEO社区/学习资源应位于表格最上方

## 代码示例
```html
<!-- 在"SEO工具/网站分析"模块添加新链接 -->
<tr>
  <td><a href="https://traffic.cv/" target="_blank">Traffic.cv</a></td>
  <td><a href="https://dash.3ue.co/zh-Hans/#/page/m/home" target="_blank">UE (Semrush/Similarweb)</a></td>
  <td><a href="https://www.semrush.com/analytics" target="_blank">Semrush Analytics</a></td>
  <td></td>
</tr>
```
